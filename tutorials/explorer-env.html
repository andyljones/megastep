An Explorer Env (Incomplete)¶

In this tutorial we’re going to take the minimal environment we studied earlier and build it into the Explorer environment.

Compared to the minimal env, the explorer env

  • Has compound RGB-depth-IMU observations.

  • Has movement with momentum.

  • Outputs rewards for exploring.

  • Resets agents when they’ve lived too long.

You can see the final explorer code here.
